# Práctica 8 – CleanCity App

Práctica final: app de reportes ciudadanos con login, almacenamiento en Firebase, subida de imágenes y mapa con marcadores.

## Cómo correr

1. Abre la carpeta **practica-08** en Android Studio.  
2. Configura tu proyecto Firebase y agrega `google-services.json`.  
3. Añade tu API Key de Google Maps en `AndroidManifest.xml`.  
4. Run.

## Requisitos

- Android Studio  
- Firebase (Auth, Firestore, Storage)  
- Google Maps SDK  
- JDK 17
