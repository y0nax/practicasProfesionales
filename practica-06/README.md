# Práctica 6 – Reproductor MP3 App

App que reproduce archivos de audio en formato MP3 usando **MediaPlayer**. Muestra metadatos y controles básicos.

## Cómo correr

1. Abre la carpeta **practica-06** en Android Studio.  
2. Ejecuta en un emulador/dispositivo con archivos MP3 disponibles.  

## Requisitos

- Android Studio  
- Kotlin 1.9+  
- Permisos de almacenamiento habilitados
