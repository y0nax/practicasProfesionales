# Práctica 1 – Hola Mundo App

Aplicación básica con **Jetpack Compose** que muestra el mensaje *“¡Hola Mundo!”* en pantalla y permite cambiar el texto al presionar un botón.

## Cómo correr

1. Abre la carpeta **practica-01** en Android Studio.  
2. Conecta un emulador/dispositivo con API 24+.  
3. Run.

## Requisitos

- Android Studio (versión reciente)  
- Kotlin 1.9+  
- JDK 17  

Notas
  - Cada práctica es independiente.
  - Algunas necesitan configuración extra (Firebase o API Key).
  - Para más detalles entra al README.md de cada carpeta.
