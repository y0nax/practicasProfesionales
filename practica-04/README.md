# Práctica 4 – GitHub App

App que consume la API de GitHub para mostrar repositorios y detalles de usuarios.

## Cómo correr

1. Abre la carpeta **practica-04** en Android Studio.  
2. Ejecuta el proyecto en un emulador/dispositivo.  

## Requisitos

- Android Studio  
- Retrofit / OkHttp  
- Internet activo
