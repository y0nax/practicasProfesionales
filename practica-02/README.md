# Práctica 2 – Lista de Tareas App

App sencilla que permite agregar y eliminar tareas usando **Jetpack Compose** y `LazyColumn`.

## Cómo correr

1. Abre la carpeta **ListaTareasApp** en Android Studio.  
2. Ejecuta en un emulador/dispositivo.  
3. Agrega tareas y pruébalas.

## Requisitos

- Android Studio actualizado  
- Kotlin 1.9+  
- JDK 17
