# Práctica 7 – Expense Tracker App

App de control de gastos con persistencia en **Room**. Permite filtrar por fechas, eliminar con swipe y exportar a CSV.

## Cómo correr

1. Abre la carpeta **ExpenseTrackerApp** en Android Studio.  
2. Ejecuta en un emulador/dispositivo.  

## Requisitos

- Android Studio  
- Room Database  
- Kotlin 1.9+
