# Práctica 3 – Clima App

App que consulta el clima en una ciudad mediante una API externa y muestra el estado en pantalla.

## Cómo correr

1. Abre la carpeta **ClimaApp** en Android Studio.  
2. Añade tu **API Key** en el archivo indicado (README del código).  
3. Run.

## Requisitos

- Android Studio  
- Retrofit + Gson  
- Internet activo
