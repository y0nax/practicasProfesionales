# Práctica 5 – Calculadora App

App de calculadora con botones numéricos y de operaciones. Usa Jetpack Compose y evalúa expresiones con librería externa.

## Cómo correr

1. Abre la carpeta **CalculadoraApp** en Android Studio.  
2. Run ▶ y prueba operaciones básicas.

## Requisitos

- Android Studio  
- Kotlin 1.9+  
- Librería exp4j incluida en dependencias
